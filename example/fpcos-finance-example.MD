# FPCOS Finance Example Skill

## PIPELINE
```
INPUT → [L0 Reality Anchor] → [L1 Axiom Gate + Market Framing]
      → [L2 System Lens] → [D1 Market Structure] → [D2 SMC Analysis]
      → [L3 Compound Mind] → [L4 Shadow Gate + Market Extensions]
      → [L5 Synthesis + Trade Structure]
```

## FPCOS CONTRACT COMPLIANCE

| Contract | Implementation |
|---|---|
| L0 Reality Anchor | Price data=KNOWN. Interpretation=INFERRED. Future=UNKNOWN. |
| L1 Kalama10 | Claims tested against price structure, not sentiment/rumor |
| L1 Ariya4 | Trend / Structure / Bias / Entry-Exit |
| L4 Shadow Gate | 5 core + Liquidity Check + Regime Check |
| L5 Confidence | Confidence%, scenario probabilities, invalidation levels |

## D1 — MARKET STRUCTURE
```
Price → Liquidity → Order Flow → Price (R loop)
Leverage   : liquidity void zones
Dead weight: ranging/choppy action
```

## D2 — SMC ANALYSIS
```
Order blocks : institutional zones
FVG          : fair value gaps
Liquidity    : buy/sell side pools
Bias         : bullish / bearish / neutral
```

## L4 SHADOW GATE — EXTENSIONS
Core: Mirror | Inversion | Blind Spot | Interest Map | Meta-Void
Domain:
- Liquidity Check: Position size executable? Liquidity at target levels?
- Regime Check: Analysis assumes which regime? What if regime shifts?
