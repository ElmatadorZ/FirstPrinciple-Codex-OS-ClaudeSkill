## Overview

The Kalama10 is FPCOS's universal proof standard.

**Core axiom:** Accept a claim ONLY when it survives all 10 rejection tests.
**Proof = Understanding.** Knowing THAT something is true ≠ knowing WHY it is true.

---

## The 10 Rejection Tests

### Test 1 — Rumor / Word-of-mouth
Pass: independently traceable to primary source, verified without transmission chain.

### Test 2 — Tradition
Pass: evidence of effectiveness under *current* conditions, not historical ones.

### Test 3 — Logic Alone
Pass: logic + observed consequence. Valid argument can still be factually false.

### Test 4 — Analogy Alone
Pass: direct evidence for the specific claim. Analogy = intuition only, not proof.

### Test 5 — Preference
Pass: preference acknowledged as preference, not presented as evidence.

### Test 6 — Authority Alone
Pass: authority + independently testable claim. Authority narrows prior, doesn't prove.

### Test 7 — Theory-Fit
Pass: alternative models considered. Evidence discriminates between them.
*(Most violated by AI: training distribution creates apparent confirmation.)*

### Test 8 — Single Source
Pass: multiple independent sources via different methods or data.
When only one source exists — state this as a limitation.

### Test 9 — Consensus Alone
Pass: consensus + traceable evidence chain. Consensus = strong signal, not proof.

### Test 10 — "Seems Reasonable"
Pass: operationally tested. Prediction derived from claim checked against observable reality.

---

## Claim Evaluation Format

```
Claim       : [state the claim]
Source type : [rumor / tradition / logic / analogy / preference /
               authority / theory / single source / consensus / intuition]
Tests failed: [list which of 10 tests it fails]
Verdict     : ACCEPT / REJECT / INFERRED (requires explicit labeling)
```

---

## The Proof = Understanding Axiom

Fully proven = can answer all four:
1. WHAT is true
2. WHY it is true (causal/mechanistic explanation)
3. UNDER WHAT CONDITIONS it is true (boundary conditions)
4. WHAT WOULD FALSIFY it (testable)

Claims that cannot answer all four = INFERRED, not KNOWN.
