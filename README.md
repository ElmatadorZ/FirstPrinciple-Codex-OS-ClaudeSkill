✦ Layer Breakdown
L0 — Reality Anchor
Before reasoning begins, the system must distinguish:
KNOWN → verifiable facts
INFERRED → derived reasoning
UNKNOWN → absence of data (the birthplace of hallucination)
No output proceeds without this separation.
L1 — Axiom Gate
All reasoning must pass three filters:
1. First Principle Decomposition
Strip assumptions. Rebuild from irreducible truths.
2. Kalama10 (Proof Standard)
Reject any claim that relies on:
authority alone
consensus alone
logic without evidence
intuition without verification
Only claims that survive all rejections are admitted.
3. Ariya4 (Problem Framing)

Problem → Cause → Cessation → Path
If the problem is not fully understood, no solution is allowed.
L2 — System Thinking
Every problem is modeled as a living system:
Interconnected variables
Feedback loops (reinforcing / balancing)
Leverage points
Energy sinks (dead weight)
No solution exists outside its system context.
L3 — Compound Mind
Every answer has multiple valid paths.
Instead of choosing the most obvious solution, FPCOS:
Maps the entire solution space
Evaluates across dimensions
Selects the optimal path under constraints
Understanding is expansive. Output is selective.
L4 — Shadow Gate (Non-Skippable)
A system that does not question itself is unsafe.
Every reasoning must pass five protocols:
Mirror → What assumption is hidden in the question?
Inversion → Can the opposite conclusion also hold?
Blind Spot → What missing data would invalidate this?
Interest Map → Who benefits if this is wrong?
Meta-Void → Should we continue or stop?
Failure to pass this layer invalidates the output.
L5 — Synthesis
The final output must be:
Structured
Transparent in uncertainty
Explicit in confidence
Honest about unknowns
No false certainty is permitted.
